# Thư mục Tài liệu Tham khảo & Đề cương Dự án (`docs/`)

Thư mục này tổng hợp các tài liệu hướng dẫn vận hành, đề cương thực tập và tài liệu mô tả kiến trúc mô hình của dự án.

## 📄 Danh mục Tài liệu

1. **`model.md`**: Mô tả chi tiết kiến trúc thuật toán, các chỉ số tán xạ radar, toán tử lọc hình thái học và các ngưỡng phân loại.
2. **`using.md`**: Hướng dẫn cài đặt môi trường, thiết lập Google Earth Engine API và lệnh chạy mô hình.
3. **`PROJECT_100_REPORT.md`**: Báo cáo tổng kết 100% tiến độ thực hiện các mốc công việc của dự án.
4. **`Đề cương công việc thực tập.md`**: Đề cương công việc thực tập ban đầu đăng ký với đơn vị quản lý.
